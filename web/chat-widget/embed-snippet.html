  • Click "+ Add Code"
  • Paste the embed code above
  • Name it "Aurora Chat Widget"

3Configure Placement

  • Set "Add Code to Pages" to "All Pages"
  • Set "Place Code in" to "Body - end"
  • Click "Apply"
